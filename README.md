# University Management System Database

## Project Overview
This project is a relational database system for a university management system. It is developed for the B103 Databases & Big Data module.

The database manages students, instructors, courses, and enrollments. It demonstrates how structured data can be stored, retrieved, and managed using SQL.

## Files Included
- `schema.sql` - creates the database tables
- `sample_data.sql` - inserts sample records into the tables
- `queries.sql` - contains SQL queries for CRUD operations, joins, and aggregations
- `ER_diagram.png` - ER diagram of the database structure
- `README.md` - project documentation

## Database Features
- Relational schema with primary and foreign keys
- Normalized table structure
- Sample data for testing
- CRUD queries
- Join queries
- Aggregation queries

## How to Run
1. Open your SQL environment (such as SQLite, MySQL, or DB Browser for SQLite).
2. Run `schema.sql` to create the tables.
3. Run `sample_data.sql` to insert sample records.
4. Run the queries in `queries.sql` to test the system.

## Example Use Cases
- View all students and courses
- Enroll students in courses
- Update student records
- Delete enrollment records
- Generate reports such as total students per course
